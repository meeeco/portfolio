2020/08 processing羊のアニメーション / Java

processingで羊を描き色々な機能をつけました
プログラミングを使って初めて作った作品です。

editor : processing
言語 : Java
制作期間：１ヶ月半

羊の作り方

耳をパタパタさせながら羊がテクテク歩きます。耳や足の動きはパーツを交互に点滅させることで表現しました。
羊は全て円をしようして描きました


クリックすると顔が和みます

画面をクリックすると元の目のパーツは黒になり新たに目とほっぺが表示されます


音楽ファイルを再生します

再生ボタンをクリックすると音楽ファイルが再生され、再度クリックすると停止します
minimライブラリを使用しました
